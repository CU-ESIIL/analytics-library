# ESIIL Analytics Library

The Analytics Library shows what you can do with environmental data once you have it.

The companion [ESIIL Data Library](https://cu-esiil.github.io/data-library/) answers: **How do I get data?**

This library answers: **What can I do with it?**

## Use This With the Data Library

Start from a dataset in the [Data Library](https://cu-esiil.github.io/data-library/), then choose an analytic workflow here based on method, data type, or domain tags.

Examples:

- Climate time series data -> forecasting, break detection, trend analysis.
- Sentinel-2 or other raster data -> cloud correction, NDVI, classification.
- Fire event data -> post-fire recovery, early-warning features, machine learning.
